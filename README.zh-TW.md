## 怎能使用
```shell
go get github.com/fast-response/fast-response
```

## 功能
- [X] JSON操作
- [X] Chunked
- [X] From-data
- [X] Cookie操作

## 協議
- [X] HTTP/1
- [X] HTTP/1.1
- [ ] HTTP/2
- [ ] HTTP/3
- [ ] WS
- [ ] SSL/TLS(需要等待`gnet`實現)
